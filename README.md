마이배민 · 음식 룰렛 (UT Prototype)

배달앱에서 메뉴를 고르는 과정을 더 쉽고 재미있게 만들기 위해 제작한 음식 추천 룰렛 웹 프로토타입입니다.
배달의민족 마이배민(MyBaemin) UI 스타일을 참고해 민트톤 비주얼, 카드형 레이아웃, 레버 인터랙션을 적용했습니다.

사용자는 레버 버튼을 눌러 룰렛을 돌리고,
랜덤으로 선택된 음식 결과를 확인한 뒤 추천 화면으로 이동할 수 있습니다.

🎯 프로젝트 목적

이 프로젝트는 다음과 같은 목적을 바탕으로 제작되었습니다.

마이배민 스타일의 UI 구조와 인터랙션 테스트

✨ 주요 기능
1. 음식 룰렛 인터랙션

레버 버튼을 클릭하면 룰렛이 회전하고, 랜덤으로 음식 메뉴가 선택됩니다.

2. 레버 버튼 애니메이션

레버 버튼은 이미지 없이 CSS만으로 구현되어 인터랙션 요소를 강화했습니다.

3. 결과 화면 이동

룰렛이 멈춘 뒤 선택된 음식 정보를 localStorage에 저장하고 menu.html 결과 페이지로 이동합니다.

4. 배민 스타일 UI

민트톤 컬러, 카드형 컴포넌트, 모바일 중심 레이아웃을 적용해 배달앱 느낌을 반영했습니다.

🖥 사용 방법
1) 웹에서 바로 사용

아래 배포 주소에서 바로 실행할 수 있습니다.

https://wlgydl12-alt.github.io/yoojihyen/

2) 로컬에서 실행

로컬 환경에서 확인하려면 간단한 HTTP 서버를 실행합니다.

python3 -m http.server

이후 브라우저에서 아래 주소로 접속합니다.

http://localhost:8000
