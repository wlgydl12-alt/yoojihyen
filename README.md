# 마이배민 · 음식 룰렛 (UT Prototype)

이 저장소는 **음식 메뉴 선택을 도와주는 룰렛 웹 프로토타입**을 담고 있습니다.
배달의민족 **마이배민(MyBaemin)** UI 스타일을 참고하여 민트톤 디자인과 인터랙션을 적용한 실험용 웹 프로젝트입니다.

사용자가 레버 버튼을 당기면 음식 룰렛이 회전하고,
선택된 메뉴 결과 페이지로 이동하도록 구현되어 있습니다.

# 🎯 프로젝트 목적

이 프로젝트는 다음 목적을 위해 제작되었습니다.

* UX/UI 인터랙션 프로토타입 제작
* 배달앱 사용자 경험 실험
* 음식 선택 과정의 재미 요소 제공
* 웹 기반 룰렛 인터페이스 구현

---

# ✨ 주요 기능

### 음식 룰렛 인터랙션

사용자가 레버 버튼을 누르면 룰렛이 회전하고 랜덤 메뉴가 선택됩니다.

### 레버 버튼 애니메이션

이미지 없이 CSS로 구현된 레버 버튼 인터랙션이 적용되어 있습니다.

### 결과 화면 이동

룰렛이 멈추면 선택된 음식 정보를 `menu.html` 페이지로 전달합니다.

### 배민 스타일 UI

배달의민족 느낌의 민트톤 UI와 카드형 레이아웃을 적용했습니다.

# 🖥 사용 방법

## 1️⃣ 웹에서 바로 사용

브라우저에서 아래 주소를 열면 바로 사용할 수 있습니다.

https://wlgydl12-alt.github.io/yoojihyen/

---

## 2️⃣ 로컬에서 실행

로컬 환경에서 확인하려면 간단한 HTTP 서버를 실행합니다.

```bash
python3 -m http.server
```

그 후 브라우저에서 아래 주소로 접속합니다.

```
http://localhost:8000
```
